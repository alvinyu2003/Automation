# Automation
Example App for UITests

## Requirements
Download latest [Xcode](https://apps.apple.com/us/app/xcode/id497799835?mt=12)

## How to run tests
- Open Xcode project
- Press cmd+u

## How to add test
- Open Xcode project
- Press play
- Go to test file/function
- Press record
- Press stop
- Add assertion
